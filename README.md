# Project Setup Guide

## Project Info

This is a modern web application built using:

* Vite
* TypeScript
* React
* shadcn-ui
* Tailwind CSS

---

## Getting Started

You can run this project locally using the following steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate into the project folder
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

---

## Editing the Project

You can edit the project using any of the following methods:

### Use Your Preferred IDE

* Open the project in your local development environment
* Make changes and push updates to your repository

### Edit Directly on GitHub

* Navigate to the file you want to edit
* Click the pencil icon (Edit)
* Commit your changes

### Use GitHub Codespaces

* Go to your repository
* Click on the **Code** button
* Open the **Codespaces** tab
* Launch a new Codespace
* Edit and push changes directly

## Deployment

To deploy this project:

* Build the project:

```sh
npm run build
```

* Preview the production build:

```sh
npm run preview
```

* Deploy using your preferred hosting platform (e.g., Vercel, Netlify, or a custom server)
